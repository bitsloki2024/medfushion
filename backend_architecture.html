Step 5 — Machine Learning Layer (ml/ folder)
🔍 anomaly.py
IsolationForest (scikit-learn)
Why: Non-parametric · handles non-normal distributions
Input: case counts time series
Output: bool[] — True = epidemic spike or data gap
📈 forecast.py
Prophet (primary) → LinearRegression (fallback)
Why Prophet: Handles seasonality + missing data
Input: historical years + case counts
Output: predicted cases ± confidence bounds
⚠️ risk.py
Risk Formula + K-Means (k=3)
Score = prevalence(40) + growth(40) + burden(20)
Labels: LOW / MODERATE / HIGH / CRITICAL
KMeans: clusters countries by risk tier
enriched data → JSON response
